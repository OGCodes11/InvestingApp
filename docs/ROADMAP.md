# InvestIQ Roadmap

## Phase 1: Foundation

- Set up monorepo and shared TypeScript contracts
- Stand up API service and Postgres
- Implement auth middleware using Firebase tokens

## Phase 2: MVP Features

- Social feed CRUD with comments and likes
- Investor profiles with follows
- Markets page with top bought/sold and news aggregation
- Course modules with progress tracking
- Badge unlocking and verification eligibility state

## Phase 3: Beta Readiness

- Direct messaging with realtime delivery
- Basic moderation tooling and abuse controls
- Error monitoring and operational dashboards
- Closed beta onboarding and feedback loops
